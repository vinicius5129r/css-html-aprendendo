Fontes Externas

Este é um exemplo de como usar fontes externas em um documento HTML.

Para usar uma fonte externa, você pode incluir um link para a fonte no cabeçalho do seu documento HTML. Por exemplo: 

<head>
  <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
</head>

Depois de incluir o link para a fonte, você pode usar a fonte em seu CSS. Por exemplo: 

body {
  font-family: 'Roboto', sans-serif;
}

Substitua 'Roboto' pelo nome da fonte que você escolheu no Google Fonts.
