Principais formatações de texto em HTML

Negrito / Destaque

Nessa frase, a palavra destaque está em negrito, ultilizando a tag <strong>, que indica que o texto tem uma importância especial(semântica).

Nessa frase, a palavra destaque está em negrito, ultilizando a tag <b>, que apenas altera a aparência do texto sem indicar importância(não semântica).

Itálico / Ênfase

Nessa frase, a palavra ênfase está em itálico, ultilizando a tag <em>, que indica que o texto deve ser enfatizado(semântica).

Nessa frase, a palavra itálico está em itálico, ultilizando a tag <i>, que apenas altera a aparência do texto sem indicar ênfase(não semântica).
